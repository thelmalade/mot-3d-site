Run it through a local server

The 3D globe and live data load as JavaScript modules, which browsers block over file://. In this folder, run one of these and open the URL it prints:

python -m http.server 8123
# then open  http://localhost:8123

Tip: double-click start.cmd in this folder to do it automatically.
